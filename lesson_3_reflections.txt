Ques1- When would you want to use a remote repository rather than keeping all your work local?
Ans- To show my work to others.
Ques2- Why might you want to always pull changes manually rather than having Git automatically stay up-to-date with your remote repository?
Ans- You have to be sure that the changes you made to your local repository should be present on github which may or may not be accessed by other people.
Ques3- Describe the differences between forks, clones, and branches. When would you use one instead of another?
Ans- Forks means to clone the repositories only on the github servers, Branches where different commits are placed in order of time. clones are done when you want to wrok on your local machine while branches lets you create new features on your project.
